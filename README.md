# Mvvm
1.Add it in your root build.gradle at the end of repositories:
	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
2.Add the dependency
	dependencies {
	   implementation 'com.github.wwangxiang:Mvvm:v0.0.1'
	}
