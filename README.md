# Mvvm
1.Add it in your root build.gradle at the end of repositories:<br>
allprojects {<br>
repositories {<br>
...<br>
maven { url 'https://jitpack.io' }<br>
}<br>
}<br>
2.Add the dependency<br>
dependencies {<br>
implementation 'com.github.wwangxiang:Mvvm:v0.0.1'<br>
}<br>

</repositories><br>
